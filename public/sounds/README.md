# Sons audio

Ce dossier contient les fichiers audio utilisés dans l'application.

## Fichier requis

### `oss.mp3`
Fichier audio qui dit "oss" (terme utilisé en jiu-jitsu brésilien pour saluer, remercier ou confirmer).

**Spécifications recommandées :**
- Format : MP3 ou WAV
- Durée : 0.5 - 1 seconde
- Volume : Normalisé (éviter les pics)
- Qualité : 44.1 kHz, 128 kbps minimum

**Où trouver un fichier "oss" ?**
1. Enregistrer votre propre voix disant "oss"
2. Utiliser une bibliothèque de sons libres (Pixabay, Freesound, etc.)
3. Générer avec un outil de synthèse vocale (TTS) si nécessaire

**Note :** Si le fichier `oss.mp3` n'est pas présent, l'application utilisera automatiquement un son synthétique en fallback.
